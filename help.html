Conversation24 File Picker Help

Use this app to connect to the SFTP server, browse files, upload new files, and copy public asset URLs.

Connect

  1. Enter the SFTP host, port, username, private key path, and remote folder.
  2. Enter the key passphrase for the current session.
  3. Click Connect and Open File List.

Browse And Copy URLs

  1. Use the search field to filter the file list.
  2. Image files can show thumbnails directly in the file list using their public asset URL.
  3. Click Copy URL next to a file to copy its public asset URL.
  4. The copied URL is shown inside the app after each action.

Upload A File

  1. Open the upload screen from the main file view.
  2. Drag a file onto the drop zone or click Choose File.
  3. Confirm the selected file appears in the upload area.
  4. Click Upload Selected File.

Troubleshooting

  • Check that the private key path points to a real file on disk.
  • Confirm the passphrase is correct for the current key.
  • Verify the remote folder exists and your account can write to it.
  • If upload fails, reconnect and try again with the same file.

Note: Settings are saved locally, but the passphrase is intentionally entered each time.
